⚗ Experimental Transcript Sources

These features are not guaranteed to be stable. They are safe to enable and can be turned off at any time. If an experimental source fails, the extension automatically falls back to the standard speech recognition path.

Use Google Meet captions for live transcript

Reads captions directly from Google Meet's on-screen caption UI when available. Automatically falls back to speech recognition if captions are not detected or stop unexpectedly. May break if Google Meet changes its interface.

Privacy

  • Meeting audio is processed entirely in memory — never written to disk.
  • Transcripts are sent directly to your chosen provider from your browser using your own API key.
  • Your API keys are stored in your local browser profile (encrypted at rest by Chrome).
  • No analytics, telemetry, or backend servers are used by this extension.
  • Session data is cleared as soon as you dismiss the meeting notes.
