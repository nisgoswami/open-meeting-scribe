Open Meeting Scribe

Settings

OpenAI API Key

Your API key is stored locally in your browser and is never sent to any server other than OpenAI. Get a key at platform.openai.com/api-keys.

Summary Model

The GPT model used to generate structured meeting notes from the transcript. Faster models cost less; more capable models produce better notes.

Privacy

  • Meeting audio is processed entirely in memory — never written to disk.
  • The raw transcript is sent directly to OpenAI from your browser.
  • Your API key is stored in your local browser profile (encrypted at rest by Chrome).
  • No analytics, telemetry, or backend servers are used by this extension.
  • Session data is cleared as soon as you dismiss the meeting notes.
